---
name: portfolio-case-builder
description: Use esta skill para estruturar páginas e cards de cases do portfólio San_Node com foco em problema, solução, processo, tecnologias e resultado.
---

Objetivo:
- Organizar projetos do portfólio San_Node em formato de case profissional.

Estrutura recomendada para cada case:
- Nome do projeto.
- Tipo de projeto.
- Contexto.
- Problema identificado.
- Solução proposta.
- Processo de desenvolvimento.
- Tecnologias utilizadas.
- Principais funcionalidades.
- Decisões de design.
- Resultado ou benefício.
- Próximos passos, quando aplicável.

Regras:
- Não inventar resultados numéricos sem dados reais.
- Quando não houver métrica real, usar resultados qualitativos.
- Evitar parecer portfólio fictício demais.
- Apresentar o projeto com clareza comercial.
- Destacar o valor entregue, não apenas a tecnologia usada.
- Manter linguagem profissional, objetiva e humana.
