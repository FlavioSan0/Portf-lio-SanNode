description: Use esta skill para criar ou ajustar backgrounds animados tech clean no portfólio San_Node, sem prejudicar performance, leitura ou responsividade.
---

Objetivo:
- Criar efeitos visuais tecnológicos sutis para fundos do portfólio San_Node.

Direção visual:
- Fundo escuro.
- Elementos abstratos.
- Linhas sutis.
- Grid discreto.
- Pontos conectados.
- Partículas leves.
- Brilho ciano controlado.
- Movimento suave e elegante.

Regras:
- Não prejudicar leitura do conteúdo.
- Não usar animação pesada.
- Não depender de bibliotecas novas sem justificativa.
- Preferir CSS, SVG ou canvas leve quando possível.
- Respeitar prefers-reduced-motion.
- Garantir boa performance no mobile.
- O background deve complementar o conteúdo, não competir com ele.
- Evitar aparência exagerada de hacker/cyberpunk.
